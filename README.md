# Cursor Effects

Interactive visual experiments that respond to cursor movement, generating dynamic and random visuals.

🔗 **Live Demo:** [mastartm.github.io/cursor-effects](https://mastartm.github.io/cursor-effects/)

---

## Effects

### ⬡ Hexagon Grid
Hover over hexagons to paint them with random colors. Each hexagon lights up once and stays.

### ◎ Cursor Ring
A circular ring that follows your cursor, shifting through hue colors with GSAP animations.

### 〜 Smoke Trail
Fluid smoke particles that follow your cursor and cycle through colors. Touch support included.

---

## Tech Stack

- Vanilla JavaScript
- HTML + CSS
- Canvas API
- GSAP
